# RedGrowth.partners

**B2B Community Marketing & Reddit Strategy Agency**

A high-performance, enterprise-grade landing page designed for rapid lead generation, flawless Core Web Vitals, and elite technical SEO.

## ⚡ Performance Metrics
This codebase has been strictly optimized to achieve top 1% performance on Google PageSpeed Insights (Mobile & Desktop):
* **LCP (Largest Contentful Paint):** < 0.5s
* **TBT (Total Blocking Time):** 0ms
* **CLS (Cumulative Layout Shift):** 0.1
* **Tracking:** Event-driven lazy loading for Google Tag Manager and Microsoft Clarity to prevent render-blocking.

## 🛠 Tech Stack
* **Markup:** Semantic HTML5 (WCAG Accessibility Compliant)
* **Styling:** Tailwind CSS v3.4 (Locally Compiled & Minified)
* **Interactivity:** Vanilla JavaScript
* **Icons:** Lucide Icons (Deferred)
* **Hosting/Deployment:** Netlify (Continuous Deployment)
